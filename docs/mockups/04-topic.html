.NET — the 5-minute primer

⏱ 5 min read·Updated Apr 2026
TL;DR A unified, cross-platform runtime for C#, F#, and VB — one SDK, many project types, runs on Windows, Linux, macOS.

What it is

.NET is Microsoft's open-source developer platform. The same SDK compiles web APIs, desktop apps, mobile apps, and background services. Releases follow a predictable annual cadence with LTS every two years.

Versions you'll meet

You'll mostly encounter .NET 6 (LTS), .NET 8 (LTS), and the latest current release. Avoid .NET Framework 4.x for greenfield projects.

Project types

$ dotnet new webapi      # REST API
$ dotnet new blazor      # Blazor app
$ dotnet new console     # CLI / job
$ dotnet run
💡 Tip: use global.json at the repo root to pin the SDK version your team uses.
⚠️ Gotcha: .NET Framework projects can't be opened on macOS — pick .NET (Core) for cross-platform teams.

Cheat sheet

Restore, build, test, publish — four commands cover 90% of daily work.
